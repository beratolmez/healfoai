# Screenshot Checklist

Before publishing the public showcase repository, verify:

- all screenshots are from the same visual language where possible
- no obviously broken localization is visible
- no placeholder or debug-only data dominates the screen
- the app icon and brand name are consistent
- at least one screenshot shows the core dashboard
- at least one screenshot shows meal input
- at least one screenshot shows history or profile depth
- at least one screenshot shows the AI/analysis aspect

Suggested final ordering:

1. dashboard
2. add meal
3. process flow
4. analysis result
5. history
6. profile
7. coach
8. light theme
